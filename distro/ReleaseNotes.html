I am happy to announce the release of WildFly-Camel-1.0.0.Alpha1

You can download the binary here: wildfly-camel-installer-1.0.0.Alpha1.jar

This is the first R5 compliant release for WildFly-8.0.0 integration.

  • Provide initial Camel subsystem
  • Camel integration with JNDI
  • Camel integration with JAX-WS
  • Camel integration with JMS
  • Camel integration with JMX
  • Define Camel context definition format for deployments

Please also have a look at the latest version of the WildFly-Camel Documentation.
