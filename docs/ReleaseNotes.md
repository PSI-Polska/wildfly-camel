Release Notes
-------------------

WildFly-Camel 6.1.0 provides Camel-2.21.1 integration with WildFly-12.0.0

This is a minor update release that adds a number of new components to the list

Additional components in the [supported set](http://wildfly-extras.github.io/wildfly-camel/#_camel_components) are:

* cmis
* ganglia
* lra
* pgevent
* tika
* wordpress
* yql

In addition to that, we also resolved a number of other [tasks and bugs](https://github.com/wildfly-extras/wildfly-camel/blob/master/docs/Changelog.md).

For details please see the [6.1.0 Milestone](https://github.com/wildfly-extras/wildfly-camel/issues?q=milestone%3A6.1.0).

Enjoy
